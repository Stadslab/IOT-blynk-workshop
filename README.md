# IOT-blynk-workshop
Stadslab IOT workshop using Blynk Library and ESP8266

This is the platform-IO version of this project.
[click here for arduino IDE version](../../tree/main)

## Hardware list
* NodeMCU v2
* HC-SR04 ultrasonic sensor
* DHT11
* Button
* RGB LED
* 10mm LED
* Breadboard
* Assorted wires
